# Controle Financeiro Pessoal

Sistema web para gerenciamento pessoal de finanças desenvolvido com HTML, CSS e JavaScript puro.

## Como usar

1. Abra o arquivo `index.html` no navegador
2. Cadastre seu salário
3. Adicione suas despesas com data, nome e valor
4. Acompanhe seu saldo em tempo real e visualize o gráfico de despesas

## Funcionalidades

- Cadastro de salário
- Cadastro, edição e exclusão de despesas
- Resumo financeiro em tempo real
- Gráfico de despesas vs saldo livre
- Persistência de dados com LocalStorage
- Interface tema escuro elegante
- Responsivo para dispositivos móveis

## Requisitos de browser

- Chrome, Firefox, Safari, Edge (qualquer versão recente)
- LocalStorage habilitado
- JavaScript habilitado

## Dados

Todos os dados são armazenados localmente no navegador usando LocalStorage. Nenhum dado é enviado para servidor.
